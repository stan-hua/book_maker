# AI BookMaker

## Quick Start
Install required dependencies:

```
pip install -r requirements.txt
```

## Create a Book

In order to create a book, run the following commands.

In Windows:
```
./make_book.bat --topic [TOPIC]
```

In Linux:
```
./make_book --topic [TOPIC]
```
